# 🎨 VELVORIA - WEB STITCH REVISION PROMPT (DESKTOP VIEWS)

**Analisis Hasil Stitch Sebelumnya:**
Hasil generate Anda sangat lengkap (80+ halaman), TAPI 90% berfokus pada tampilan `_mobile` (mobile view). Karena Velvoria adalah Web E-commerce, kita **sangat membutuhkan tampilan DESKTOP** (layout grid lebar, sidebar, hover state).

**Cara Pakai:**
Copy semua text di bawah garis putus-putus ini dan paste ke prompt Web Stitch Anda.

----------------------------------------------------------------------

**CONTEXT:** 
Lanjutkan project Velvoria (Premium Multi-Vendor E-Commerce). Design system sebelumnya sudah bagus (Minimalist Luxury, Deep Navy #1A1F3A, Rose Gold #E8B4A0, Cream #FFF8F3, Playfair Display & Inter).

**TASK:** 
Sekarang, buatkan varian **DESKTOP (Width minimal 1280px)** untuk halaman-halaman utama. Jangan buat versi mobile lagi. Fokus pada layout layar lebar dengan pemanfaatan whitespace yang mewah.

**HALAMAN YANG HARUS DIBUAT (DESKTOP ONLY):**

1. **Desktop Mega Navigation & Footer**
   - Navbar lebar dengan dropdown mega-menu mewah untuk kategori.
   - Search bar besar di tengah, icon cart/wishlist di kanan.
   - Footer lebar dengan 4 kolom link.

2. **Desktop Homepage (Luxury Storefront)**
   - Hero section layar penuh dengan parallax effect. Layout split text kiri, gambar model/produk besar di kanan.
   - Section "Featured Collections" dengan grid asimetris (bukan carousel biasa).
   - "Top Vendors" showcase dengan 4 kolom.

3. **Desktop Product Catalog / Shop Page**
   - **Kiri (25%):** Sticky Sidebar untuk Filters (Kategori, Harga, Brand, Vendor) dengan checkbox custom.
   - **Kanan (75%):** Product Grid (3 kolom).
   - Tambahkan animasi hover pada product card (zoom gambar lambat, muncul tombol Add to Cart).

4. **Desktop Product Detail Page (Split Layout)**
   - **Kiri (60%):** Galeri foto produk besar dengan thumbnail di samping kirinya.
   - **Kanan (40%):** Sticky product info. Judul besar, rating, form pilih varian/size yang elegan, tombol Add to Cart lebar (Rose Gold).

5. **Desktop Shopping Cart & Checkout (Split Layout)**
   - **Kiri (65%):** List item di Cart ATAU Form Checkout (Shipping, Payment).
   - **Kanan (35%):** Sticky Order Summary box (warna background abu-abu sangat muda/Cream) agar ringkasan order selalu terlihat saat scroll.

**KEY REQUIREMENT UNTUK DESKTOP:**
- Tambahkan **Hover States** (karena di desktop ada kursor mouse).
- Pastikan jarak (padding/margin) antar section sangat besar (100px - 160px) untuk kesan Luxury & Editorial.
- Jangan gunakan hamburger menu, tampilkan semua navigasi utama di atas.
